##  Getting Started

###  Prerequisites

Before getting started with realtime-map, ensure your runtime environment meets the following requirements:

- **Programming Language:** Go version 1.24.6
- **Package Manager:** Go modules
- **Container Runtime:** Docker

**Build from source:**
